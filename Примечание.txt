День 1.
— Подготовим наше веб-приложение, сделаем простую верстку.
— Получим элементы со страницы
— Познакомимся с API - база данных фильмов
— Получим API ключ 
— Подключим API к нашему веб приложению через XMLHttpRequest

День 2. 
— Данные которые мы получили будем выводить на страницу
— Изучим Promise и fetch

День 3. 
— Разберем данные которые получаем, выводим на страницу постер и сможем получать полное описание фильма 
— Реализуем несколько запросов, в зависимости от типа видео (сериал/фильм)
— Если у фильма/сериала есть трейлер в youtube на русском языке, то на него будет ссылка

День 4.
— Для чего нужен JavaScript
— Что нужно, чтобы учить JavaScript
— Насколько перспективен этот язык
— План изучения JavaScript
